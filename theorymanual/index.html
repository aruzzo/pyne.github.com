Theory Manual¶

Welcome to the PyNE Theory Manual! This manual will guide you through the mathematics and physics underlying PyNE.

The following describe methods implemented natively in PyNE:

  • Spatial Solvers
  • nucname
  • Decay
  • Data
  • Source Sampling
  • Variance Reduction
  • Rigorous Two Step
  • Binning

For information on methods for which PyNE provides an interface but does not implement natively, please refer to the manuals associated with those methods.
